如何切换到极速模式

国产双核浏览器：

如果您当前正在使用的已经是 QQ 浏览器、360 安全浏览器、百度浏览器、搜狗浏览器、2345 浏览器等主流国产双核浏览器，请按照下面的教程切换到该浏览器的“极速模式”：

  1. 1. 打开浏览器浏览网页，浏览器的上方有个小闪电的图标。

  2. 2. 点击这个图标下边就出现一个选择的对话框。

  3. 3. 点击切换到兼容模式即可，切换完成后可以使用同样的办法给切换回去。

微软 Microsoft Edge 浏览器：

要回到在不用 Internet Explorer 模式的情况下进行浏览，可关闭当前标签页，再在 Microsoft Edge 中打开新的标签页。

如果打开后仍为 IE 模式下浏览，请按照下面的教程切换到该浏览器的“极速模式”（方法同国产双核浏览器的类似）：

  1. 1. 打开浏览器浏览网页，浏览器的上方有个灰色的 Microsoft Edge 的图标。

  2. 2. 点击这个图标下边就出现一个选择的对话框。

  3. 3. 点击切换到兼容模式即可，切换完成后可以使用同样的办法给切换回去。
